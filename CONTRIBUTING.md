### Running the app locally

Install package, then `npm run start`. This will start the app on http://localhost:3000/ which you can visit in the browser.

Go to Miro.com, developer settings, then apps.

There should be one app registered called Clyde Recommends and another one called Clyde Recommends Development. The Clyde Recommends Development should point to the public url http://localhost:3000/
